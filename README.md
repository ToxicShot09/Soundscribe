# Soundscribe Translator

## Overview
Soundscribe Translator is a project designed to transcribe and translate audio files using advanced machine learning models, including OpenAI's Whisper.

## Features
- Audio file upload and playback.
- Transcription services powered by Whisper.
- Integration with Supabase for backend services.
- Responsive UI built with React and Tailwind CSS.

## Project Structure
- **backend/**: Contains Python scripts for transcription and model handling.
- **src/**: Frontend code including components, hooks, and services.
- **supabase/**: Configuration for Supabase integration.

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd soundscribe-translator
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the backend server:
   ```bash
   python backend/main.py
   ```
2. Start the frontend development server:
   ```bash
   npm run dev
   ```
